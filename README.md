# NMCH – Проверка номеров на наличие в Telegram  
**Проверка регистрации телефонных номеров в Telegram – быстро, точно, масштабируемо**

Незаменимый инструмент для сбора качественной аудитории, генерации лидов и маркетинга в Telegram в 2025–2026 годах.  
Превращайте сырые базы телефонных номеров (из любых источников) в реальные Telegram-цели.


### 🌐 Доступно на нескольких языках
- English – [английская версия](https://github.com/telegram-prime/telegram-numbers-checker)  
- Русский – эта страница  
- 简体中文 – [NMCH Checker CN](https://github.com/telegram-prime/telegram-numbers-checker-CN)

### ⚙️ Основные возможности

| Возможность                          | Описание                                                                    |
|--------------------------------------|-----------------------------------------------------------------------------|
| Проверка регистрации в Telegram      | Определяет, зарегистрирован ли номер в Telegram                             |
| Поддержка входных данных             | Любые списки номеров (рекомендуется международный формат)                   |
| Типы аккаунтов                       | TDATA, SESSION (Telethon/Pyrogram), SESSION+JSON (автоопределение)          |
| Режим без аккаунтов                  | Проверка только через прокси (без загрузки профиля)                         |
| Многопоточность и высокая скорость   | Одновременная работа с несколькими аккаунтами → огромная производительность |
| Поддержка прокси                     | HTTP / SOCKS5 — почти все провайдеры; можно назначить уникальный прокси на поток |
| Защита от флуда и банов              | Многоуровневый контроль флуда, автоматическое исключение ограниченных/забаненных аккаунтов |
| Очистка телефонной книги             | Опция очистки контактов аккаунта перед проверкой (безопаснее и чище)        |
| Лимит на аккаунт                     | Установка максимального количества номеров на аккаунт за цикл               |
| Сортировка и отчёты                  | Valid / Invalid → отдельные файлы; экспорт в TXT, CSV, XLSX                 |
| Богатые экспортируемые данные        | Номер • Telegram ID • Username • Имя • Последний онлайн • Premium • Фото • Био • День рождения • Канал • Сторис |
| Отчёт по производительности          | Сколько номеров проверил каждый аккаунт (по сессиям)                       |
| Подробные логи по сессиям            | Полный лог активности сохраняется отдельно — идеально для отчётов клиентам  |
| Отключение GUI-логов                 | Выключение консольного лога для сверхбольших объёмов (ускорение работы)     |
| Работа без прокси                    | Возможна (для тестов / низкорисковых сценариев)                             |
| Привязка к железу                    | Один ПК на ключ; бесплатный сброс через менеджера при смене оборудования    |
| Бесплатные пожизненные обновления    | Регулярные улучшения и совместимость со слоями Telegram                     |
| Онлайн-поддержка                     | Быстрая помощь напрямую от разработчиков                                    |
